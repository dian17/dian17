笔记面向对象

课程任务：理解JavaScript基本面向对象思想。JavaScript中对象的组成，用工厂方式去创建一个对象
学习资料：建议参考妙味课堂视频面向对象内容（1，2，3节）；
课程笔记：
必修内容：
  一、什么是面向对象(Object-Oriented，OO )？
           1、理解对象
                    让我们来复习一个之前学过创建对象的两种方式：
自定义对象的方法（new方法）：（校准说法）
var person = new Object();
//属性
person.name = "DongYang";
person.age = 24;
person.job = "Software Engineer";
//方法
person.sayName = function() {
     alert(this.name);
};
person.sayName();
alert(person.job);
对象字面量的方法：
var person = {
     //属性
     name: "DongYang";
     age: 24;
     job = "Software Engineer";
     //方法
     sayName: Function() {
         alert(this.name);
     }
};


                          
           2、什么是面向对象?
           3、面向对象是一种通用的思想，并非只有编程中能用，任何事情都能用。
二、JavaScript中面向对象
           1、面向对象编程（OOP）的特点
                 抽象：抓住核心问题
                 封装：不考虑内部实现，只考虑功能使用
                 继承：从已有对象上继承出新的对象
                            多重继承
                            多态
           2、对象的组成
                  方法——函数：过程、动态的
                  属性——变量：状态、静态的
三、第一个面向对象程序
            1、为对象添加方法和属性
                    this详解，事件处理中this的本质
                           -window
                           -this—函数属于谁
                    不能在系统对象中随意附加方法、属性、否则会覆盖已有的方法、属性
                    object对象
四、工厂方式
            1、什么是工厂方式
                     原料      加工      出厂
             2、工厂方式的
                   用构造函数创建一个类
                    什么是类，对象（实例）：模具和零件
